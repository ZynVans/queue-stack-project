# 📌 Data Structure Mini Project 1

## Queue & Stack Customer Service System

---

## Description

This project is a simple simulation of a customer service system using fundamental data structures: Queue and Stack.

* Queue is used to manage customers waiting in line (FIFO – First In First Out)
* Stack is used to store the history of served customers (LIFO – Last In First Out)

This program helps demonstrate how data structures are applied in real-life scenarios.

---

## Objectives

* Understand Queue and Stack concepts
* Implement Queue and Stack in C programming
* Apply problem-solving using data structures
* Build a structured and readable program

---

## Features

* Add customer to queue
* Serve customer (remove from queue)
* Store served customer into history (stack)
* Display current queue
* Display service history

---

## Data Structures Used

### 1. Queue (FIFO)

* First customer entered will be served first
* Implemented using array
* Uses `front` and `rear`

### 2. Stack (LIFO)

* Last served customer will appear first in history
* Implemented using array
* Uses `top`

---

## Program Structure

### Queue Functions

* `initQueue()` → initialize queue
* `enqueue()` → add data
* `dequeue()` → remove data
* `isEmptyQueue()` → check if empty

### Stack Functions

* `initStack()` → initialize stack
* `push()` → add to stack
* `displayStack()` → show history
